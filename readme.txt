1) Hướng dẫn chạy project
Clone git project: https://github.com/hoanglong1698/MyCoins.git
Chuyển đến thư mục Frontend: cd Frontend
Chạy lệnh npm install và npm start


2) Mô tả hệ thống
Ứng dụng mô tả một hệ thống tiền điện tử MyCoin đơn giản
Khi chạy ứng dụng, mặc định balance = 10
Ta có thể chuyển tiền đến một địa chỉ ví khác với số tiền nhỏ hơn hoặc bằng balance đang có
Sau khi chuyển thì ta phải mining để thêm block vào mạng lưới và nhận được phần thưởng mining reward
Tùy độ khó mà mining sẽ nhanh hay chậm
Ta có thể chỉnh độ khó và phần thưởng khi mining
Nhấn vào một block để xem các giao dịch trong một block
Nhấn vào địa chỉ một ví để xem chi tiết ví (bao gồm balance và lịch sử giao dịch)
